Button
Buttons allow users to interact with the application.
Primary Button
The primary buttons are used to interact with the primary actions to a website
Outline Button
The outline button style removes all background images or colors from a button and gives it a lighter look.
Floating Button
A floating action button (FAB) performs the primary, or most common, action on a screen.
Link Button
The LinkButton control has the same appearance as a HyperLink control, but has the same functionality as a Button control
Icon Button
Icon buttons are useful where interface space may be limited. If an icon represents the action well, users can sometimes recognize them quicker than reading text.
