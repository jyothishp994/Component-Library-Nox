# Component-Library-Nox

# Nox UI

NOx UI is a component library with basic css components that are easily usable to build a web application.


## Installation


<link rel="stylesheet" href="https://nox-ui.netlify.app/css/style.css" />


## Components Included
- Avatar
- Alert
- Badge
- Button
- Card
- Image
- Input
- Text Utilities
- Navigation
- Grid
- Modal
- Rating
- Toasts
- Lists
